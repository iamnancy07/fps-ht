# FPS Shooter Game

A multiplayer FPS shooter game built with HTML5.

## Features

- Multiplayer gameplay (or play against bots)
- Lobby system for player interaction
- Shop for buying weapons and abilities
- Settings menu
- Real-time combat

## Getting Started

Open `index.html` in your browser to play.

## Development

To run a local server:
```
npm start
```

Then open `http://localhost:8080`

## Deployment

This project is ready to deploy on Vercel. Simply push to GitHub and connect to Vercel for automatic deployments.
